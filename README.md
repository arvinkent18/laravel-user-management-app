## Laravel User Management App
> Laravel 6.2 User Management App is use for managing users from the back end which uses AdminLTE template combine with VueJS

``` bash
# Install Dependencies
composer install

# Generate key
php artisan key:generate

# Run migrations
php artisan migrate
```
